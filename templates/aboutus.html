About Our Project

Phishing Awareness & Detection System is a security-focused project designed to educate users about phishing attacks and help them identify malicious websites using intelligent URL analysis, real-world attack patterns, and interactive learning.

Project Overview

This project focuses on spreading awareness about phishing attacks and protecting users from online fraud. It includes a URL Scanner that analyzes websites, educational pages explaining different phishing techniques, real-world examples, CAPTCHA-based login, OTP email verification, and a chatbot assistant to explain security concepts.
