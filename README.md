# PHISHING-AWARENESS-AND-DETECTION-SYSTEM-
Phishing Awareness &amp; Detection System is a Flask-based cybersecurity web app that educates users about phishing attacks and scans URLs using heuristic analysis. It includes login/signup with OTP, CAPTCHA, URL scanner, phishing awareness modules, chatbot support, and modern UI to promote cyber safety.
# Phishing Awareness & Detection System

A web-based system designed to educate users about phishing attacks and detect suspicious URLs using heuristic analysis.

## Tech Stack
- Python (Flask)
- HTML, CSS, JavaScript
- Flask-Mail
- JSON Database

## Features
- Secure Login & Signup with OTP
- CAPTCHA Verification
- URL Scanner (Safe / Suspicious)
- Phishing Awareness Module
- Interactive Chatbot
- Modern UI with animations

## How to Run
1. pip install -r requirements.txt
2. python app.py
3. Open http://127.0.0.1:5000
